# 使い方（CSV → 座標更新 → INP書き換え）

## 実行方法

1. `input/` にCSVを入れる
2. `inp/` に `.inp` を1つだけ入れる
3. `main.exe` を実行
4. 処理完了後、`Enterで終了` の表示が出たら Enter を押して終了

## 目的

- `input` フォルダ内の CSV（任意の名前）を読み込み、変位から `X_new, Y_new, Z_new` を作成して `output/output.csv` に保存します。
- `inp` フォルダ内の `.inp` ファイル（1つだけ）を読み込み、`*NODE` セクションの節点座標を `output/output.csv` の `Node Label, X_new, Y_new, Z_new` に合わせて置き換え、更新済みINPを出力します。
- 実行時に、重い処理（読み込み・計算・書き込み等）の実行時間を表示します。

---

## フォルダ構成

`.exe`（または `main.py`）と同じ階層に、次のフォルダを置いてください。

(メインフォルダ)
├─ main.exe
├─ _internal/
│ └─ （削除不可）
├─ input/
│ └─ （読み込みたいCSVファイルを1つだけ入れる：任意の名前でOK）
├─ inp/
│ └─ （書き換え対象のinpファイルを1つだけ入れる：任意の名前でOK）
└─ output/
  └─ （自動生成：結果が出力されます）

### 重要（inpフォルダのルール）

- `inp/` 内の `.inp` は **1つだけ**にしてください。  
  0個だとエラー、2個以上だと「複数あるので1つにしてください」とエラーになります。

## 出力ファイル

実行後、output/ に次のファイルが生成されます。

- output/output.csv
- 入力CSVに X_new, Y_new, Z_new を付加したもの（全行）
- output/output_check.csv
- 先頭100行の確認用（軽量）
- また、INPは inp/ に更新版が出力されます
